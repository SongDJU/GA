# 이지켐 총무 자산관리

계약 및 전표 관리 시스템

## 주요 기능

### 🔐 사용자 관리
- 비밀번호 없이 사용자 선택 로그인
- 관리자/일반사용자 권한 구분
- 관리자만 사용자 추가 가능

### 📝 전표 관리
- 적요명, 금액, 부가세액, 계정과목명, 반복일자 관리
- 처리완료 체크 기능 (체크 시 메일 알림 제외)
- 엑셀 업로드/다운로드

### 📋 계약 관리
- 계약명, 카테고리, 계약업체, 금액, 만료일 관리
- 카테고리: 임대계약, 유지보수, 서비스계약, 구매계약, 차량, 기타
- 첨부파일 업로드 지원
- 엑셀 업로드/다운로드

### 📧 메일 알림
- 매일 오전 9시 자동 발송 (KST)
- 전표: 해당 월 미처리 전표 알림
- 계약: 만료 45/30/20/10/3/2/1일 전 알림
- 이쁜 HTML 메일 템플릿

### 💾 기타
- 휴지통 (복구 가능)
- 대시보드 (이번 달 전표, 만료 임박 계약)

## 기술 스택

- **Framework**: Next.js 16 (App Router)
- **UI**: TailwindCSS + Shadcn/UI
- **Database**: SQLite + Prisma ORM
- **인증**: NextAuth.js
- **스케줄러**: node-cron
- **메일**: Nodemailer
- **엑셀**: xlsx

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 환경변수 설정
cp .env.example .env
# .env 파일 수정

# 데이터베이스 마이그레이션
npx prisma migrate dev

# 초기 데이터 시드
npm run db:seed

# 개발 서버 실행 (포트 9090)
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 실행 (cron 스케줄러 포함)
npm start
```
